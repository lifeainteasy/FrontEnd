## DOCTYPE은 무엇을 하나요?
---
DOCTYPE은 document type의 약어입니다.
DOCTYPE은 항상 DTD(Document Type Definition)와 관련됩니다.
DTD는 특정 문서가 어떻게 구성되어야 하는지 정의합니다
